# 행성: 지구 (Text RPG Game)

고등학생 때 C언어를 처음 배우고 만든 텍스트 기반 RPG 게임입니다.

## 프로젝트 소개

이 프로젝트는 C언어 기초를 배우면서 만든 첫 번째 게임 프로젝트입니다. 
구조체, 파일 입출력, 포인터 등 C언어의 기본 개념들을 실습하며 제작했습니다.

## 게임 특징

- 레벨 시스템과 경험치 획득
- 몬스터 전투 시스템
- 아이템 및 포션 관리
- 스텟 분배 시스템
- 상점 시스템
- 슬롯머신 미니게임
- 데이터 저장/불러오기

## 게임 플레이

### 메인 메뉴
1. **탐험** - 몬스터와 싸워 경험치와 골드 획득
2. **정보확인** - 캐릭터 레벨, HP, MP, 스텟 확인
3. **아이템 확인** - 보유 중인 포션과 마법 확인
4. **상점** - 포션 구매 (마법 상점은 미구현)
5. **스텟 설정** - 레벨업으로 얻은 포인트 분배
6. **슬롯머신** - 골드를 걸고 2배로 불리기
7. **이용가이드** - 게임 설명
8. **종료** - 게임 저장 후 종료
9. **초기화** - 게임 데이터 초기화

### 탐험 지역
- **몬스터의 초원** (Lv.1~) - 초보자용 사냥터
- 늑대의 굴 (미구현)
- 엘프의 숲 (미구현)
- 뱀파이어의 성 (미구현)

### 스텟 시스템
- **힘** - 공격력 증가
- **민첩** - 속도 증가
- **방어** - 받는 데미지 감소
- **지력** - 마법 데미지 증가
- **크리티컬** - 치명타 확률 증가
- **운** - 슬롯머신 당첨 확률 증가

## 실행 환경

- **운영체제**: Windows
- **개발 환경**: Visual Studio
- **언어**: C
- **필요 파일** (실행 폴더 기준 상대 경로 — `game/` 에서 실행):
  - `test/data.txt` - 유저 데이터 (저장 파일, 없으면 자동 생성)
  - `test/stat.txt` - 스텟 데이터 (저장 파일, 없으면 자동 생성)
  - `test/monsters.txt` - 몬스터 정보 (게임 콘텐츠)
  - `test/magicspell.txt` - 마법 정보 (게임 콘텐츠)

## 설치 및 실행

> 데이터 파일은 더 이상 `C:\test\` 절대 경로를 쓰지 않고, 실행 폴더 기준 `test/` 상대 경로를 사용합니다. (refactor 브랜치 0단계)

1. 저장소를 클론한 뒤 `game/` 폴더로 이동합니다. (`test/` 하위 폴더에 txt 파일이 이미 들어 있습니다)
2. 컴파일합니다.
   - GCC(MinGW): `gcc -std=c11 -Wall -Wextra -o game.exe code.c`
   - 또는 Visual Studio에서 `game/code.c` 빌드
3. **반드시 `game/` 폴더에서** 실행합니다 (`test/` 폴더가 보이는 위치).
   - `./game.exe`

## 문제점

이 프로젝트는 학습 목적으로 만들어진 초기 작품으로, 다음과 같은 문제점들이 있습니다:

- 파일 경로가 `C:\test\`로 고정되어 있음

- 에러 처리 부족
- 코드 중복 다수
